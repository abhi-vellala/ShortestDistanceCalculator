Shortest Distance Calculator

This demo uses Travelling Salesman Problem approach to calculate the shortest distance between all nodes. Travelling Salesman Problem is one of the most popular ways of determining the shortest path between given points. This is extensively used in Computer Science research in optimizing a process or a program to reduce computation complexity and achieve more control. More information can be found in Wikipedia page
